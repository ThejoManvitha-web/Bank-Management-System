A console-based Bank Account Management System developed in C++ that allows
users to create accounts, securely log in, perform banking transactions,
and maintain transaction history using file-based persistence.

 Features :
- Create a new bank account
- User authentication using account number and password
- Deposit money into the account
- Withdraw money from the account
- Check current account balance
- View account details
- View complete transaction history
- Persistent storage using files
